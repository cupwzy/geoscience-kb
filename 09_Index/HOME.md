## Operating Rhythm

1. 先把未经整理的东西丢进 [[00_Inbox/Inbox|Inbox]] 或当天 Daily Note。
2. 每周把 Inbox 中的条目移动到学科 MOC、项目页、论文页或数据页。
3. 每个重要判断都链接到证据：论文、数据集、图件、井、地震剖面或实验记录。
4. 项目推进时优先写“判断链”：观察 -> 解释 -> 不确定性 -> 下一步验证。
