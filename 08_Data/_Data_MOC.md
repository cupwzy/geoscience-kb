# Data MOC

## Data Objects

- Wells
- Logs
- Seismic Volumes
- Horizons and Faults
- Rock Physics Tables
- Training Labels
- Model Outputs

## Data Hygiene

- 记录坐标系、单位、深度域/时间域、采样率和版本。
- 记录处理历史，不只记录文件名。
- 对 ML 数据，单独记录标签来源和划分策略。
