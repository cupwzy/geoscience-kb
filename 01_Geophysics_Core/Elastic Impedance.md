# Elastic Impedance

## Core Idea

弹性阻抗把角度依赖反射信息转换成类似阻抗的属性表达，常用于连接叠前地震、AVO 响应和储层预测。它适合做相对趋势解释，但需要明确角度范围、频带、标定井和反演约束。

## When Useful

- 对比不同角度下的岩性或流体敏感性。
- 作为叠前反演结果和岩石物理模板之间的桥梁。
- 为机器学习模型提供更接近物理意义的输入特征。

## Risks

- 角度范围过窄时可解释性下降。
- 处理流程中的振幅不一致会被放大。
- 与井上弹性参数拟合良好，不等于空间预测可靠。
