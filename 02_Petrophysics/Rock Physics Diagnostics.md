# Rock Physics Diagnostics

## Purpose

岩石物理诊断用于把测井、岩心、流体和地震响应放到同一个解释框架中，判断某个地震属性是否真的能指示储层性质。

## Diagnostic Views

- Vp-Vs、AI-Vp/Vs、Lambda-Rho vs Mu-Rho。
- 孔隙度、泥质含量、含水饱和度与弹性参数交会图。
- 分岩性、分层位、分压力系统的统计分布。

## Interpretation Rule

先确认分离度，再谈预测。若目标类别在井上弹性空间本身不可分，地震属性或 ML 模型通常只会给出表面上更平滑的不可分结果。
