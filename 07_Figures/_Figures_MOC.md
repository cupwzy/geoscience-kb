# Figures MOC

## Figure Types

- 井震标定图
- 岩石物理交会图
- 地震剖面解释图
- 属性切片
- ML 结果图
- 文献复现图
- Mermaid 流程图与概念关系图
- Canvas 知识图谱
- Excalidraw 手绘推导图和地质示意图

## Rule

每张关键图都应记录来源、数据版本、生成脚本或软件、解释结论和可复用场景。
